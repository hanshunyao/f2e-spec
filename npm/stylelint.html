# hansy-f2e-spec-stylelint-config

TIP

hansy-coding CSS 规范

支持配套的 stylelint 可共享配置 (opens new window)。

# 安装

需要先行安装 stylelint (opens new window) 和 stylelint-scss (opens new window)：

npm install hansy-f2e-spec-stylelint-config stylelint stylelint-scss --save-dev

# 使用

在 .stylelintrc 中继承本包:

{
  "extends": "hansy-f2e-spec-stylelint-config"
}
